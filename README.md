# Analyse de Ventes avec Python et Pandas

## Description
Système complet d'analyse de données de ventes construit en Python.

## Fonctionnalités
- Chargement et exploration de données CSV
- Calcul du chiffre d'affaires par ville, catégorie et vendeur
- Identification du meilleur vendeur
- Export des résultats en CSV et Excel
- Génération automatique de rapport

## Technologies utilisées
- Python 3
- Pandas
- CSV
